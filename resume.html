Caloni Trasporti S.r.l.

Project Manager · 2003 - 2005

Analysis, development and managment of the VisualBasic.Net program GestioneTrasporti. The main goal of the project was to let the client print their own adhesive labels to the packages they were going to send with Caloni Trasporti S.r.l. Also, the program tracked the personal history of the client and the data transfer between the client and Caloni's IBM AS400.

The main feature achieved was that the program was extremely customizable: it could print with every thermal printer, receiving every kind of text record data and sending every kind of different report to Caloni just by changing few variables in external files.

With the growing amount of clients gained using GestioneTrasporti, I started a part-time collaboration with Caloni as Project Manager to continue to update the software with all the clients needs.

I covered also the role of the network administrator, taking charge of installing and configuring an internal UNIX server for network security.
